## 😎 О ComfyLauncher

ComfyLauncher - это инструмент для удобного, быстрого, и "лёгкого" запуска portable-версий ComfyUI.

Standalone-версия Comfy имеет свой собственный загрузчик, что очень удобно в использовании. Поэтому мне хотелось создать схожий опыт запуска для Portable-версии вместо открытия в браузере по умолчанию.

Я использую разные билды ComfyUI для разных задач, например - один специально для работы с WAN, другой для тестирования новых фич, третий для работы с генерацией изображений и так далее. При этом, я не хочу использовать один универсальный билд для всего, чтобы избежать возможных конфликтов. Думаю, многим людям, особенно работающим в продакшене, знаком этот подход - универсальность не всегда хороша и стабильна. Поэтому мой выбор пал на сборку отдельных портативных билдов для каждого типа задач.

При этом, меня не очень устраивало, что портативка ComfyUI всегда открывается в браузере по умолчанию. Мой браузер довольно тяжелый, в нём много вкладок, и я хотела использовать для Comfy отдельный пустой браузер, но даже в таком случае при пуске сервера Comfy открывает браузер по умолчанию. Конечно, это не является большой проблемой, но требует лишних движений. И тем более это актуально, когда у вас каждый мегабайт в RAM на счету.

Поэтому я приняла решение разработать отдельный загрузчик, который будет удобен в практическом применении. Основные фичи и идеи программы описываю ниже.

## 🎨 Функции

- **Менеджер запуска нескольких сборок.**  
  Добавляйте и управляйте несколькими портативными сборками ComfyUI с запуском в один клик. 
Для каждой сборки можно задать собственные флаги запуска, отображаемое имя и иконку.  
&nbsp;

- **Лёгкий встроенный браузер.**  
  Потребляет мало оперативной памяти — особенно важно для средних по мощности машин или ресурсоёмких задач. Не содержит лишних компонентов, характерных для обычных браузеров, поэтому запускается быстро.
На 30% меньше потребление RAM по сравнению с обычным Chrome (в режиме ожидания) — протестировано на Windows 10 и Windows 11.  
&nbsp;

- **Возможность показать или скрыть окно CMD.**
Если окно терминала, работающее в фоне и захламляющее панель задач, раздражает — его можно скрыть.
&nbsp;


- **Встроенная консоль.**  
  При отключенном окне CMD Launcher имеет вывод из CMD в специальную консоль UI, кнопка которой появляется автоматически. Таким образом после скрытия терминала вы не теряете возможность детального мониторинга.  
&nbsp;

- **Кнопки быстрого доступа и состояния сервера, которые часто нужны при работе.**
  - Открыть директорию **Output** и директорию **ComfyUI**
  - **Refresh UI** - обновить UI
  - **Restart** - запуск и перезапуск сервера
  - **Stop** - полная остановка сервера  
&nbsp;

- **Поддержка стандартных тем ComfyUI**, чтобы интерфейс выглядел целостным.
- **Индикация работы сервера** - Online, Offline, Restart
- **И другое.**

## ⚒ Установка

- Перейдите в раздел **Releases** и скачайте необходимый релиз.
- Распакуйте архив в желаемую директорию.
- Запустите ".exe" и используйте!
- Убедитесь, что у вас установлена среда выполнения Microsoft WebView2. Если нет, пожалуйста, загрузите и установите [Evergreen Bootstrapper](https://developer.microsoft.com/en-us/microsoft-edge/webview2).
